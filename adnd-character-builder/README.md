# AD&D Character Builder

A new project folder for an AD&D character builder. This directory contains starter files and a simple structure to begin development.

## Structure

- `src/index.html` - HTML entry point
- `src/main.js` - Main JavaScript file
- `src/styles.css` - Basic styles
